Vær i sentrum!

I denne leksjonen skal du få lære noen triks som du kan bruke for å sentrere innholdet horisontalt på nettsiden din!
Hovedteknikken for å løse dette handler om å forstå hvordan div -taggen fungerer. Noen ganger må lage en div inne i div som omslutter (wrapper) elementet vi skal sentrere. For å forstå sentrering ordentlig må vi forstå følgende;

  • display attributtet
  • marginer
  • størrelse
